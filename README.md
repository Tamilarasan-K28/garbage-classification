# ♻️ Garbage Classification Using AI

This project uses **Deep Learning (CNN)** to classify waste images into six categories:

- 🟤 Cardboard  
- 🟡 Glass  
- ⚙️ Metal  
- 📄 Paper  
- 🧴 Plastic  
- 🗑️ Trash

Built using:
- 🧠 TensorFlow / Keras
- 🌐 Streamlit (for web interface)
- 📦 Python
